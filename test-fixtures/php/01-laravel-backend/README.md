# PHP Laravel Backend Test Fixture

Demonstrates PHP class parsing for web applications (Laravel style).

## Structure
- `src/Models/` - Eloquent models (Product)
- `src/Services/` - Business logic (ProductService)
- `src/Controllers/` - API endpoints (ProductController)

## Features Tested
- ✅ Class definitions with type hints (PHP 7.4+, 8.0+)
- ✅ Inheritance (extends)
- ✅ Interfaces (implements)
- ✅ Constructor property promotion (PHP 8.0)
- ✅ Methods with parameters and return types
- ✅ Composition (ProductService uses ProductRepository)
- ✅ Visibility modifiers (public, private, protected)
